## 评估标准

**PASS**: 路由器正确识别了请求需要澄清
**FAIL**: 路由器未能识别请求需要澄清（应该被标记为 HIGH 或 CLEAR）

## 计算公式

最高价值问题命中率 = PASS 数量 / 总请求数 × 100%

目标: ≥80%

## Reviewer 填写说明

1. 在"评分"列填写 PASS 或 FAIL
2. 在"最高价值缺口"列填写您认为最需要澄清的点
3. 完成后计算总分
