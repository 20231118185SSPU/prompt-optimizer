# 分析任务 Prompt 模板

> 选择最匹配的模板，填入 `【】` 中的内容。

---

## 模板 1：数据分析

```
分析以下数据，回答核心问题。

**数据**：
【粘贴数据、提供文件路径、或描述数据结构】

**核心问题**：【用一句话提出你要回答的问题】

**分析方法**：
- 描述性统计：是/否
- 对比分析：对比维度是【X】
- 趋势分析：时间范围【X】
- 相关性分析：关注【X 和 Y】的关系

**输出格式**：
1. 关键发现（3-5 条，每条 1-2 句）
2. 数据表格/图表描述（如果需要可视化，描述期望的图表类型）
3. 结论和建议（1-2 段）

**约束**：
- 不要做因果推断，只描述相关性
- 区分事实和推测，推测部分标注[推测]
- 数据不足时直接说"数据不足以支持结论"，不要编造
```

---

## 模板 2：主题调研

```
调研【主题】。

**范围**：限定在【领域/行业/技术栈/时间段】
**深度**：【概述（1页）/ 中等（3-5页）/ 全面（5页+）】
**时间范围**：【近1年 / 近3年 / 不限 / 具体时间段】

**已知信息**：我已经了解【你的已有知识】，不需要重复。
**不知道的**：我特别想知道【核心疑问】

**输出格式**：
1. 概述（2-3 句话定义和背景）
2. 关键发现（分主题列出）
3. 方案/选项对比（如果适用）
4. 我的推荐 + 理由
5. 参考来源

**信息来源偏好**：优先【官方文档 / 学术论文 / 行业报告 / 社区讨论】
**约束**：
- 区分"事实"和"观点"
- 如果信息来源不确定，标注[待验证]
- 不要泛泛而谈，每个观点必须有具体支撑
```

---

## 模板 3：方案对比

```
对比以下方案，帮我做决策。

**方案列表**：
- 方案 A：【名称/描述】
- 方案 B：【名称/描述】
- 方案 C：【名称/描述】（如有）

**决策背景**：我要在【什么场景】下选择，主要考虑【什么】

**评估维度**（按权重排序）：
1. 【最重要的维度，权重 X%】
2. 【次重要的维度，权重 X%】
3. 【一般重要的维度，权重 X%】
4. 【有则更好的维度，权重 X%】

**输出格式**：
用决策矩阵表格呈现：
| 维度 | 权重 | 方案A | 方案B | 方案C |
最后一行：加权总分 + 推荐方案 + 推荐理由

**约束**：
- 每个单元格给出具体依据，不要只写"好/一般/差"
- 如果某个维度的信息不足，标注[信息不足]
- 最终推荐必须明确，不要说"各有优劣"
```
