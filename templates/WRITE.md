# 写作任务 Prompt 模板

> 选择最匹配的模板，填入 `【】` 中的内容。

---

## 模板 1：文章/博客

```
为【目标受众】写一篇关于【主题】的【类型：技术博客/观点文/教程/分析文】。

**核心论点**：【一句话说清楚文章要证明什么或传达什么】
**篇幅**：【字数】字
**语气**：【轻松/专业/学术/口语化/幽默】
**结构**：
1. 开头：【切入点——问题/故事/数据/反常识】
2. 主体：【要点1】→【要点2】→【要点3】
3. 结尾：【总结/行动号召/开放问题】

**关键要求**：
- 必须包含：【数据/案例/代码/引用】
- 不要出现：【营销话术/特定术语/过于简化的类比】
- 风格参考：【某篇文章/某个作者的风格】

**示例句**（语气参考）：
"【写一句你期望的语气和风格的示例】"
```

---

## 模板 2：邮件

```
写一封邮件。

**收件人**：【角色/关系——上级/客户/同事/陌生人】
**目的**：【核心诉求——申请/通知/催促/感谢/道歉/介绍】
**语气**：【正式/半正式/友好/坚定】
**篇幅**：【短（<100字）/中（100-300字）/长（300+字）】

**必须传达的信息**：
1. 【核心信息1】
2. 【核心信息2】
3. 【核心信息3（如有）】

**行动号召**：收件人看完后需要做什么？【回复确认/点击链接/安排会议/无行动】

**约束**：
- 不要：【过度客套/使用附件/提及某话题】
- 要：【直接切入主题/给出截止日期/附上参考链接】
```

---

## 模板 3：创意写作

```
创作一篇【类型：短篇小说/诗歌/剧本/文案/宣传语】。

**主题**：【核心主题】
**风格类似**：【参考作品/作者】
**基调**：【温暖/阴郁/悬疑/讽刺/激昂/平静】
**篇幅**：【字数/行数/段落数】

**关键元素**：
- 角色：【角色设定（如有）】
- 场景：【地点和时间（如有）】
- 核心冲突/张力：【什么矛盾推动叙事】

**约束**：
- 不要用：【特定的表达方式/陈词滥调/某种结构】
- 要做到：【具体的技术要求——如"通感"、"留白"、"节奏变化"】

**第一句参考**：【如果你想指定开头方向】
```
